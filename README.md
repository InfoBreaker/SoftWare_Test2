## 任务需求介绍
**这里是软件工程实验二的介绍。**
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;选择一个合适的软件系统，对该系统进行认真分析，撰写对应的软件需求规格说明书，具体要求：
 1. 参考相应的模板，对应的内容必须完整。
